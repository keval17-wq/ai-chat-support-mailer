The Project Roadmap: An Evolving Journey

Phase 1: Support Mailer (Current)

The core SDK is live, providing a robust solution for generating and sending AI-drafted support emails. It's a focused, niche tool that solves a specific problem effectively.

Phase 2: Enhanced Functionality (Planned)

The next stage focuses on adding versatility. Planned features include custom HTML templates, multi-language support, and the ability to handle file attachments for more complex support cases.

Phase 3: The Generic Engine (Future)

The long-term vision is to transform the SDK into a generic "prompt-to-channel" engine. This will allow developers to use custom prompts and deliver content to various platforms like Slack, Discord, or Teams.

The SDK is designed to grow in capability, moving from a specialized tool to a flexible communication platform. This chart visualizes the planned expansion of features in each phase.
